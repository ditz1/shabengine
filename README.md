# SHABFIGHT
### shabibi studios

### Installation

clone SHABFIGHT repository to your pc:
---
```bash
git clone [Repository URL]
cd [download-directory]
cd build
```

**By default, the Makefile is configured for Windows.**

### Building the Game

#### For Windows

Open PowerShell and run:

```powershell
make
./shabgame.exe
```

#### For Mac

Open Terminal and run:

```bash
make mac
./shabgame
```
